# 用golang开发电商项目
准备用一个月时间用golang开发单商户的电商系统，作为学go语言的成果
前端ui模仿小米官网
2020/1/17 ~ 2020/2/17

## 技术栈 
- web [gin](https://github.com/gin-gonic/gin)
- orm [xorm](https://github.com/go-xorm/xorm)
- 数据库 mysql
- 缓存   redis
- 搜索   elastic search
- 使用docker部署
- 后端管理系统模板 [vue-element-admin](https://github.com/PanJiaChen/vue-element-admin)
- bootstrap 4
- jquery 3

持续更新
